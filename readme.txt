BP Reorder Tabs lets you easily reorder BuddyPress Profile and Group tabs, and sub-tabs, from the WordPress admin.

== Description ==

Easily reorder the tabs (navigation) and sub-tabs on BuddyPress Profiles and Groups, and set any tab as the default one users will land on. You can even reorder tabs added via plugins.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'BP Reorder Tabs'
3. Activate BP Reorder Tabs from your Plugins page.

= From WordPress.org =

1. Download BP Reorder Tabs.
2. Upload the 'bp-reorder-tabs' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, etc...)
3. Activate BP Reorder Tabs from your Plugins page.

= Configuration =

Visit 'Settings > BP Reorder Tabs' to configure plugin options.
